# Bingo Board

A pixel-style bingo board for personal goals. Mark off quests, complete rows/columns/diagonals to score Bingos, and listen to music while you play.

## Features

- **Goal Bingo** — Create a board from your own goals (or use samples). Tap to mark complete, long-press to edit. Win by completing 1+ Bingos or a full blackout.
- **Cozy Tunes** — Built-in music player: play tracks from the `music/` folder, upload MP3s, or add YouTube links. Minimizable player with playlist.
- **Pixel look** — Gradient background and simple pixel-inspired UI.


## Project structure

- `index.html` — Main app and music player UI
- `css/styles.css` — Styles
- `js/game.js` — Bingo game logic
- `js/music-player.js` — Cozy Tunes player
- `js/background.js` — Background canvas
- `music/` — MP3 files and `music-list.json`
